# 简易五子棋

一个基于 Java 控制台的双人五子棋游戏，支持棋盘绘制、落子判定、胜负判断及音效反馈。

## 项目简介

本项目是一个 **10×10 棋盘** 的双人五子棋游戏，玩家通过控制台输入坐标进行落子。游戏会交替提示玩家 A 和玩家 B 落子，并实时更新棋盘，判断胜负或平局。同时，游戏为落子、非法操作和胜利提供了简单的音频反馈。

## 功能说明

- ✅ 双人轮流落子（A：○，B：■）
- ✅ 坐标合法性校验（边界检查、重复落子检查）
- ✅ 胜利条件判断（水平、垂直、对角线五子连珠）
- ✅ 平局判定（棋盘占满且无人胜利）
- ✅ 音效支持（落子音效、非法操作音效、胜利音效）

## 游戏规则

### 棋盘说明

- 棋盘大小为 **10 × 10**（0 ~ 9 行，0 ~ 9 列）
- 行号和列号均在棋盘边缘标注

### 落子规则

- **玩家 A 先手**，棋子为 `○`（白色圆圈）
- **玩家 B 后手**，棋子为 `■`（黑色方块）
- **输入格式**：`行号 列号`（例如 `5 5` 表示第 5 行第 5 列）

**非法输入包括：**

- 坐标超出 0 ~ 9 范围
- 在已有棋子的位置重复落子
- 输入非数字字符

### 胜利条件

在以下任意方向上**连续出现 5 个相同棋子**即获胜：

| 方向 | 说明 |
|------|------|
| 水平方向 | 同行连续 5 列 |
| 垂直方向 | 同列连续 5 行 |
| 对角线方向（45°） | 右上至左下斜线 |
| 对角线方向（135°） | 左上至右下斜线 |

### 平局条件

当棋盘被完全占满且没有任何一方达成胜利条件时，判定为**平局**。
